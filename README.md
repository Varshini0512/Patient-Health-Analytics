# Patient Health & Hospital Analytics

### Project Overview
This project analyzes hospital patient data using all 10 lab experiments.

### Techniques Used
 1: Correlation Heatmap
 2: Logistic Regression
 3: Linear Regression  
 4: Random Sampling
 5: Z-Test Hypothesis Testing
 6: NumPy Array Operations
 7: NaN Handling
 8: Financial Analysis
 9: Patient Metric Scoring
 10: Transaction Analysis

### Dataset
200 patients, 10 features including Age, BMI, Blood Pressure, Cholesterol, Heart Disease

### How to Run
1. Open Google Colab
2. Upload Patient_Health_Analytics_MiniProject.ipynb
3. Runtime → Run All
